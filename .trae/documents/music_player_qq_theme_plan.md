# 音乐播放器 QQ 音乐风格改造计划

## 需求概述

将当前简陋的音乐播放器主题改造为参照 QQ 音乐网页版的排版风格，同时将番茄钟功能自然地融入到主题中，不再显得突兀。

## 当前状态分析

### 现有实现（musicPlayerTheme.vue）
- 简单的三部分布局：顶部时间+状态、中央专辑封面+进度条、底部控制按钮
- 番茄钟作为核心功能直接展示，显得过于突出
- 背景为简单的渐变色彩
- 只有基础的播放控制按钮

### QQ 音乐网页版布局特点
1. **顶部导航栏**：Logo、搜索框、导航菜单
2. **左侧边栏**：音乐馆、我的音乐、歌单列表
3. **中间主内容区**：推荐内容、歌单、歌曲列表
4. **右侧边栏**：歌词、评论
5. **底部播放控制栏**：歌曲信息、播放控制、音量、播放列表

## 改造方案

### 整体布局（QQ音乐风格三栏+底部播放器）

```
┌─────────────────────────────────────────────────────────┐
│  顶部导航栏（Logo + 搜索 + 导航菜单）                   │
├──────────┬──────────────────────────┬───────────────────┤
│          │                          │                   │
│ 左侧边栏 │     中间主内容区          │   右侧边栏        │
│ 歌单列表 │     歌曲列表/歌单详情     │   番茄钟信息      │
│          │                          │   （融入歌词区）  │
│          │                          │                   │
├──────────┴──────────────────────────┴───────────────────┤
│  底部播放控制栏（封面+歌名+控制+音量+列表）             │
└─────────────────────────────────────────────────────────┘
```

### 番茄钟融合方案

将番茄钟融入**右侧边栏的歌词/信息区域**，伪装成"专注模式歌词"或"播放状态信息"：
- 标题："专注模式 · 正在播放"
- 内容显示：当前状态（专注/休息）、剩余时间、进度
- 风格：类似歌词滚动展示，文字居中，半透明背景
- 不使用显眼的番茄钟图标，改用音乐相关的视觉元素

### 页面各部分详细设计

#### 1. 顶部导航栏
- 左侧：QQ音乐 Logo（图标 + 文字）
- 中间：搜索框（带搜索图标和占位文字）
- 右侧：导航菜单（推荐、排行榜、歌单、歌手）
- 风格：深色背景，绿色主题色（#31c27c）

#### 2. 左侧边栏
- 我的音乐（我喜欢、最近播放、本地音乐）
- 歌单列表（专注歌单、休息歌单、白噪音等模拟歌单）
- 每个歌单有封面图标和名称
- 当前播放的歌单高亮显示

#### 3. 中间主内容区
- 顶部：歌单封面 + 歌单信息（名称、标签、播放数、播放按钮）
- 下方：歌曲列表（序号、歌名、歌手、专辑、时长）
- 当前播放的歌曲高亮显示，带播放动画图标
- 歌曲列表可滚动

#### 4. 右侧边栏（番茄钟融合区）
- 标题："歌词"（实际是番茄钟信息）
- 内容区：
  - 当前状态：专注中 / 休息中（用音乐术语表述，如"主歌" / "副歌"）
  - 剩余时间：大字体显示
  - 进度条：类似歌词进度
  - 下一段预告："下一首：休息 5分钟"
- 风格：居中文字、渐隐效果、类似歌词展示

#### 5. 底部播放控制栏
- 左侧：歌曲封面 + 歌曲名 + 歌手 + 收藏/喜欢按钮
- 中间：
  - 播放模式（随机/顺序/单曲循环）
  - 上一首 / 播放暂停 / 下一首
  - 进度条 + 时间显示
- 右侧：
  - 音量控制
  - 播放列表按钮
  - 全屏按钮
- 风格：深色半透明背景

## 验证步骤

1. 整体布局是否符合 QQ 音乐网页版风格
2. 番茄钟信息是否自然融入，不突兀
3. 各区域比例是否协调
4. 滚动、hover 等交互是否正常
5. TypeScript 类型检查: `npx tsc --noEmit`

## 注意事项

1. **番茄钟融合度**：不要出现"番茄"、"Pomodoro"等字样，用音乐术语替代
2. **视觉统一**：主题色使用 QQ 音乐绿 #31c27c
3. **布局比例**：左侧边栏约 200px，右侧边栏约 280px，底部播放器约 70px
4. **模拟数据**：使用周杰伦的热门歌曲作为歌单和歌曲列表数据
5. **性能**：纯静态展示，不需要真实播放功能
