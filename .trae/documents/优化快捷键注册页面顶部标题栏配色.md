# 快捷键注册页面顶部标题栏样式优化计划

## 问题分析

当前 `registershortcut/index.vue` 的 `.page-header` 样式存在以下问题：

1. **渐变背景过于强烈**：使用 `--header-gradient-from` → `--header-gradient-to` 紫蓝渐变，视觉上过于夸张
2. **阴影过重**：`box-shadow: 0 4px 20px rgba(0, 0, 0, 0.15)` 使其看起来像"英雄横幅"
3. **尺寸过大**：64px 高度 + 32px padding，占用过多空间
4. **白色文字对比度问题**：在某些深色主题下白色文字可能不够突出

## 设计参考

参考项目中其他页面的设计模式：
- **windowmode/index.vue**：使用 `section-title` + CSS 变量配色，简洁优雅
- **about/index.vue**：卡片式 `card-header` 使用渐变但更克制

## 修改方案

将顶部标题栏从"夸张渐变横幅"改为"简洁卡片标题栏"，与页面其他部分（快捷键卡片、底部提示卡片）风格统一。

### 设计效果

| 属性 | 修改前 | 修改后 |
|------|--------|--------|
| 背景 | 紫蓝渐变 | 卡片背景色 + 边框 |
