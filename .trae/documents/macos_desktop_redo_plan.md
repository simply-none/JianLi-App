# macOS 桌面模式改造计划

## 需求概述

将当前 macOS 桌面模式中的番茄钟从显眼的小组件改为自然融入的应用窗口，同时优化整体排版，使其更符合真实 macOS 桌面效果。

## 当前状态分析

### 现有实现（macOSDesktop.vue）
- 顶部菜单栏（基本正确）
- 中间两个小组件卡片（番茄钟、日期），半透明毛玻璃风格
- 底部 Dock（基本正确，但用 emoji 图标不够精致）
- 背景是简单的深蓝灰渐变
- 番茄钟作为独立小组件太显眼，不符合 macOS 桌面习惯

### macOS 桌面典型元素
1. **桌面背景**：macOS 风格的渐变/风景壁纸
2. **桌面图标**：右侧排列（Mac 习惯图标在右边）
   - Macintosh HD / 此电脑
   - 文件夹
   - 废纸篓
3. **顶部菜单栏**：苹果 logo + 当前应用菜单 + 状态栏图标 + 时间
4. **应用窗口**：桌面上打开的窗口，带交通灯按钮（红黄绿）
5. **底部 Dock**：应用图标栏，带悬浮放大效果
6. **番茄钟融入**：作为一个打开的应用窗口存在，而不是悬浮小组件

### 番茄钟融入方案

将番茄钟设计为一个**标准的 macOS 应用窗口**：
- 顶部有交通灯按钮（红黄绿：关闭/最小化/最大化）
- 标题栏居中显示"番茄钟"
- 窗口内容展示番茄钟信息
- 窗口有圆角、阴影、毛玻璃背景
- Dock 上有对应图标（下方圆点表示运行中）
- 这样番茄钟就像桌面上一个普通应用，不再突兀

### 详细设计

#### 1. 桌面背景
- macOS 风格的渐变壁纸（深色模式风格）
- 多层渐变模拟 macOS Monterey/Ventura 风格

#### 2. 桌面图标（右侧排列）
- Macintosh HD（此电脑）
- 文档文件夹
- 废纸篓
- 图标在右侧，从上到下排列
- 每个图标有图标 + 文字标签

#### 3. 顶部菜单栏
- 保持现有结构，优化样式
- 左侧：苹果 logo + 访达 + 文件 + 编辑 + ...
- 右侧：电池 + WiFi + 搜索 + 控制中心 + 时间
- 毛玻璃半透明效果

#### 4. 番茄钟应用窗口
- 位置：桌面中央偏左
- 大小：约 360px × 320px
- 标题栏：
  - 左侧：红黄绿交通灯按钮
  - 中间："番茄钟"标题
- 内容区：
  - 状态 + 轮次
  - 大字体时间
  - 进度条
  - 控制按钮
- 样式：圆角、毛玻璃背景、阴影

#### 5. 底部 Dock
- 优化图标样式（使用 Lucide 图标替代 emoji）
- 保持悬浮放大效果
- 番茄钟图标下方有运行指示器（小圆点）
- 毛玻璃背景
