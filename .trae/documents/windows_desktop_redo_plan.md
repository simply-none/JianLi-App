# Windows 桌面模式改造计划

## 需求概述

将当前简陋的 Windows 桌面模式改造为严格参照真实 Windows 桌面效果的设计，同时让番茄钟自然融入，不再显得突兀。

## 当前状态分析

### 现有实现（windowsDesktop.vue）
- 顶部任务栏 + 底部任务栏应用图标（布局有误，Windows 任务栏在底部）
- 中间两个小组件卡片（番茄钟、日期），白色背景太显眼
- 背景是简单的蓝色渐变
- 整体布局不符合 Windows 桌面规范

### Windows 桌面典型元素
1. **桌面背景**：Windows 11 风格的默认壁纸
2. **桌面图标**：左侧排列（此电脑、回收站、文档等）
3. **底部任务栏**：Windows 11 风格，应用图标居中
   - 左侧：开始按钮、搜索、任务视图
   - 中间：固定应用图标
   - 右侧：系统托盘（天气、网络、音量、时间日期）
4. **应用窗口**：桌面上打开的窗口，带标题栏和控制按钮
5. **开始菜单**：点击开始按钮弹出（可选）

### 番茄钟融入方案

将番茄钟设计为一个**打开的应用程序窗口**：
- 带有标准的 Windows 窗口标题栏（图标 + 标题 + 最小化/最大化/关闭按钮）
- 窗口可以有轻微阴影和圆角（Windows 11 风格）
- 窗口内展示番茄钟信息（状态、时间、进度条）
- 任务栏上有对应的应用图标（高亮表示正在运行）
- 这样番茄钟就像桌面上一个普通应用，不再突兀

### 详细设计

#### 1. 桌面背景
- Windows 11 风格的渐变壁纸（蓝色系渐变 + 光晕效果）
- 或者使用模拟的默认壁纸效果

#### 2. 桌面图标（左侧）
- 此电脑
- 回收站
- 文档
- 每个图标有图标 + 文字标签
- 支持 hover 高亮效果

#### 3. 底部任务栏（Windows 11 风格）
- 高度约 48px
- 毛玻璃半透明效果
- 左侧区域：开始按钮（Windows 徽标）、搜索、任务视图
- 中间区域：固定应用图标（文件资源管理器、Edge、设置、番茄钟等）
  - 番茄钟图标高亮（有下划线表示正在运行）
- 右侧系统托盘：
  - 天气图标 + 温度
  - 网络图标
  - 音量图标
  - 电池图标
  - 时间日期（两行：时间 + 日期）

#### 4. 番茄钟应用窗口
- 位置：桌面中央偏上
- 大小：约 360px × 280px
- 标题栏：
  - 左侧：番茄钟图标 + "番茄钟"文字
  - 右侧：最小化、最大化、关闭按钮
- 内容区：
  - 状态标签（专注中/休息中）
  - 大字体时间显示
  - 进度条
  - 简洁的设计，符合应用窗口风格
- 窗口阴影、圆角（Windows 11 风格）

#### 5. 模拟数据
- 桌面图标：此电脑、回收站、文档
- 任务栏应用：文件资源管理器、Edge浏览器、番茄钟、设置、商店
- 系统托盘：天气 26°C、WiFi、音量、电池、时间日期

## 验证步骤

1. 整体布局是否符合 Windows 11 桌面风格
2. 番茄钟是否自然融入为应用窗口
3. 任务栏位置和样式是否正确（底部、毛玻璃）
4. 各元素比例是否协调
5. TypeScript 类型检查: `npx tsc --noEmit`

## 注意事项

1. **任务栏位置**：Windows 任务栏在底部，不是顶部
2. **番茄钟融入**：以应用窗口形式存在，不再是悬浮卡片
3. **视觉风格**：Windows 11 风格（圆角、毛玻璃、居中图标）
4. **背景**：使用 Windows 11 风格的渐变壁纸效果
5. **简洁真实**：模拟数据要简洁，突出桌面的真实感
