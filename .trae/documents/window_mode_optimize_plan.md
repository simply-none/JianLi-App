# 窗口模式设置页面优化计划（卡片点击式 + 自定义）

## 一、方案核心思路

使用 **卡片点击** 的方式为主，同时在每项最后添加 **自定义选项**，兼顾快速选择和灵活配置。

## 三、卡片设计细节

### 1. 位置选择卡片（3x3 网格 + 自定义）

| 位置 | 卡片内容 | 代表方向 |
|------|----------|----------|
| 左上 | ↖ | top-left |
| 中上 | ↑ | center-top |
| 右上 | ↗ | top-right |
| 左中 | ← | center-left |
| 中心 | ● | center |
| 右中 | → | center-right |
| 左下 | ↙ | bottom-left |
| 中下 | ↓ | center-bottom |
| 右下 | ↘ | bottom-right |
| 自定义 | ⚙ 自定义 | 点击后输入坐标 |

**自定义交互**：点击自定义卡片后，弹出输入框让用户输入 X/Y 坐标

### 2. 尺寸选择卡片（预设 + 自定义）

| 尺寸 | 适用场景 |
|------|----------|
| 108×81 | 最小紧凑模式 |
| 200×100 | 标准模式（默认） |
| 300×150 | 大号模式 |
| 自定义 | 点击后弹出宽高输入框 |

**自定义交互**：点击自定义卡片后，弹出输入框让用户输入宽度和高度

### 3. 间隙选择卡片（预设 + 自定义）

| 间隙 | 说明 |
|------|------|
| 10px | 紧贴边缘 |
| 20px | 较小间隙 |
| 30px | 标准间隙（默认） |
| 50px | 较大间隙 |
| 自定义 | 点击后弹出输入框 |

**自定义交互**：点击自定义卡片后，弹出输入框让用户输入间隙值

## 六、交互设计

### 1. 位置选择交互
- 点击任意位置卡片，立即选中
- 点击自定义卡片，弹出坐标输入框
- 选中卡片高亮显示

### 2. 尺寸选择交互
- 点击尺寸卡片，立即选中
- 点击自定义卡片，弹出宽高输入框
- 选中卡片显示对勾

### 3. 间隙选择交互
- 点击间隙卡片，立即选中
- 点击自定义卡片，弹出间隙输入框
- 选中卡片数字加粗

### 4. 自定义弹窗交互
- 点击自定义卡片，弹窗从下方滑出
- 输入完成后点击确定，自动更新配置
- 点击取消或外部区域，关闭弹窗
