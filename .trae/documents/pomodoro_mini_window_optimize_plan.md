# 番茄钟小窗口布局优化计划

## 一、仓库调研结论

### 当前代码结构

**[pomodoroMiniWindow/index.vue](file:///C:/cod/electron-vite-vue/src/views/pomodoroMiniWindow/index.vue)**
- 使用简单的 Flex 布局，展示工作/休息状态
- 包含：状态小圆点、"倒计时"标签、倒计时数字
- 支持窗口拖拽和鼠标穿透功能
- 通过 IPC 接收主窗口同步的数据

### 当前问题分析

| 问题 | 影响 |
|------|------|
| 布局过于简单 | 信息展示不够丰富，用户体验不佳 |
| 状态区分不明显 | 仅用小圆点，视觉效果差 |
| 缺少进度可视化 | 用户无法直观了解当前阶段进度 |
| 样式基础 | 缺乏现代感和美观度 |
| 倒计时不够醒目 | 核心信息不够突出 |
| 缺少主题色区分 | 工作/休息状态视觉差异小 |

## 二、优化方案

### 1. 新布局设计

**整体布局结构：**
```
┌──────────────────────────────┐
│  ┌──────┐  工作中            │
│  │  ○   │  距离休息          │
│  └──────┘  00:23:45          │
│  ████████████████░░░░░░░░░░  │
│        75%                   │
└──────────────────────────────┘
```

**布局特点：**
- 状态图标 + 状态文字组合，清晰区分工作/休息
- 进度条可视化展示当前阶段进度
- 倒计时放大显示，更加醒目
- 添加状态主题色，增强视觉区分

### 2. 视觉优化

**工作状态（绿色主题）：**
- 状态图标：圆形绿色脉冲动画
- 背景：半透明绿色渐变
- 进度条颜色：绿色系

**休息状态（蓝色主题）：**
- 状态图标：圆形蓝色脉冲动画
- 背景：半透明蓝色渐变
- 进度条颜色：蓝色系

### 3. 新增功能

- **进度条**：根据当前时间与目标时间的比例，显示可视化进度
- **状态动画**：状态切换时添加平滑过渡动画，状态图标脉冲效果
- **更好的信息层次**：状态 → 标签 → 倒计时 → 进度条

## 四、技术实现细节

### 1. 进度计算

```javascript
const progress = computed(() => {
  const now = moment().valueOf();
  const status = curStatusC.value?.value;
  
  if (!sysData.value) return 0;
  
  let start, end;
  
  if (status === 'work') {
    start = sysData.value.startWorkTime;
    end = start + sysData.value.workTimeGapUnit * sysData.value.workTimeGap;
  } else {
    start = sysData.value.closeWorkTime;
    end = start + sysData.value.restTimeGapUnit * sysData.value.restTimeGap;
  }
  
  const total = end - start;
  const current = now - start;
  
  if (total <= 0) return 0;
  return Math.max(0, Math.min(100, (current / total) * 100));
});
```

### 3. 样式设计要点

- 使用 CSS 渐变背景区分状态
- 状态图标使用脉冲动画
- 进度条使用圆角和阴影效果

- 保持背景半透明，不遮挡窗口内容

## 五、潜在依赖与风险

### 依赖
- Element Plus（可选，用于图标）
- Vue 3 Composition API（已存在）
- moment（已存在）

### 风险点

| 风险 | 影响 | 解决方案 |
|------|------|----------|
| 时间计算精度 | 进度条可能出现负数或超过100% | 添加边界检查 |
| 动画性能 | 小窗口频繁更新可能卡顿 | 使用 CSS 动画而非 JS |
| 主题切换 | 状态切换时颜色变化可能突兀 | 添加过渡动画 |
| 鼠标穿透 | 新布局可能影响穿透功能 | 确保穿透区域不受影响 |
