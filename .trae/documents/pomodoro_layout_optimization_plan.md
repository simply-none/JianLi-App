# 番茄钟设置项布局优化计划

## 1. 问题分析

### 当前布局问题
- **状态卡片**：状态标签和时间信息布局过于简单，视觉层次不够清晰
- **时间卡片**：
  - 数值显示过大（48px），与其他元素比例不协调
  - 调整按钮和单位选择器位置分散，视觉不平衡
  - 卡片内部元素排列不够紧凑
- **整体间距**：各个元素之间的间距不一致，缺乏统一的视觉节奏
- **视觉层次**：缺乏主次分明的设计，所有元素看起来都一样重要

### 具体问题点
1. 状态卡片的状态点太小，不够醒目
2. 时间卡片的数值和单位分离，不够直观
3. 调整按钮太小，不易点击
4. 单位选择器与数值区域分离，操作不便

---

## 2. 优化方案

### 2.1 状态卡片优化
- 使用圆形状态指示器替代简单的圆点
- 增加状态描述文字，提供更丰富的状态信息
- 时间信息使用更紧凑的布局

### 2.2 时间卡片优化
- 将时间卡片改为横向布局，数值和控制按钮在同一行
- 数值和单位合并显示，更加直观
- 调整按钮使用更大的尺寸，增加点击区域
- 单位选择器移到卡片右侧，与数值区域对齐

### 2.3 视觉层次优化
- 增加主标题区域的装饰线条
- 时间卡片使用渐变背景增强视觉效果
- 添加微妙的阴影和圆角
- 统一所有卡片的内边距和间距

## 5. 风险与注意事项

### 风险点
1. **功能兼容性**：修改布局可能影响原有交互逻辑
2. **响应式适配**：新布局可能在小屏幕上显示异常

### 验证步骤
1. 确认时间调整功能正常工作
2. 确认单位切换功能正常
3. 确认立即生效按钮正常
4. 验证响应式布局在不同屏幕尺寸下的显示效果
