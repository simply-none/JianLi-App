# 浏览器搜索页面设计方案

## 1. 概述

在 layout 布局下新建一个浏览器风格的搜索页面，提供多标签页管理和搜索功能。

## 2. 当前状态分析

### 项目结构
- **路由配置**: `src/router/index.ts` — 使用 `layoutRouters` 数组定义 layout 下的子路由
- **布局组件**: `src/components/layout.vue` — 提供 top/left/main/bottom 插槽
- **页面组件**: `src/views/` 下各页面使用 `index.vue` 文件
- **状态管理**: 使用 Pinia store，如 `useWindowMode.ts`

### 现有页面模式
- 页面位于 `src/views/xxx/index.vue`
- 使用 `<layout-vue>` 包装，`<template #main>` 定义主内容区
- 使用 `el-` 系列组件（Element Plus）

## 3. Proposed Changes

### 3.1 新增路由配置

**文件**: `src/router/index.ts`

```typescript
// 在 RouteNames 中添加
BROWSER: "browser",

// 在 layoutRouters 中添加
{
  path: "/browser",
  name: RouteNames.BROWSER,
  component: () => import("@/views/browser/index.vue"),
  meta: {
    title: "浏览器",
    icon: Browser, // 需要添加图标
  },
},
```

### 3.2 新增页面组件

**文件**: `src/views/browser/index.vue`

页面结构：
```
┌─────────────────────────────────────────────┐
│  标签栏 + 功能区 (top)                      │
│  [标签1] [标签2] [+]    [搜索引擎选择] [设置] │
├─────────────────────────────────────────────┤
│                                             │
│              内容体 (main)                  │
│         - 新标签页（搜索框）                 │
│         - 或网页内容预览                     │
│                                             │
└─────────────────────────────────────────────┘
```

**功能**:
1. **标签栏**: 支持多标签页、新建标签、关闭标签
2. **搜索引擎选择**: 下拉选择框，默认百度，可选 Google/Bing/Baidu/DuckDuckGo
3. **搜索框**: 中央大搜索框，回车或点击搜索按钮在新标签页打开结果
4. **新标签页**: 显示搜索框和一些快捷链接

### 3.3 搜索引擎配置

**支持的搜索引擎**:
| 名称 | URL 模板 |
|------|----------|
| 百度 | `https://www.baidu.com/s?wd={query}` |
| Google | `https://www.google.com/search?q={query}` |
| Bing | `https://www.bing.com/search?q={query}` |
| 搜狗 | `https://www.sogou.com/web?query={query}` |
| 360 | `https://www.so.com/s?q={query}` |
| DuckDuckGo | `https://duckduckgo.com/?q={query}` |

### 3.4 状态管理

**文件**: `src/store/useBrowser.ts` (新建)

管理以下状态：
- `tabs`: 标签页列表
- `activeTabId`: 当前激活的标签页 ID
- `defaultEngine`: 默认搜索引擎
- `engineList`: 搜索引擎列表

## 4. 实现细节

### 4.1 标签页数据结构

```typescript
interface Tab {
  id: string;
  title: string;
  url: string;  // 新标签页时为 'newtab'
  isNewTab: boolean;
}
```

### 4.2 搜索引擎数据结构

```typescript
interface SearchEngine {
  label: string;      // 显示名称
  value: string;      // 唯一标识
  searchUrl: string;  // 搜索 URL，{query} 占位符
  icon: string;       // 可选的图标
}
```

### 4.3 核心功能

1. **新建标签**: 点击 `+` 按钮或 `Ctrl+T`，创建新标签页
2. **关闭标签**: 点击标签上的 `×` 按钮，删除标签
3. **切换标签**: 点击标签切换
4. **搜索**:
   - 输入框输入关键词
   - 回车或点击搜索按钮
   - 在新标签页打开搜索结果
5. **搜索引擎切换**: 下拉框选择，选择后保存到 store

   - 搜索引擎切换正常
   - 搜索后新标签页打开正确
