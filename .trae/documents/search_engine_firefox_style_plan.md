# 搜索引擎模式 Firefox 风格改造计划

## 需求概述

将当前搜索引擎模式（searchEngine.vue）改为严格参照 Firefox 主页搜索模式的设计，同时让番茄钟自然融入，不再显眼突兀。

## 当前状态分析

### 现有实现（searchEngine.vue）

当前布局问题：
1. 顶部导航栏（仿百度风格，不是 Firefox 风格）
2. 中央搜索框 + Logo
3. 中间快捷链接 + 状态卡片（番茄钟太显眼）
4. 底部版权信息
5. 整体是百度/谷歌搜索风格，不是 Firefox 主页风格

### Firefox 主页（新标签页）特点

Firefox about:newtab 布局：
```
┌─────────────────────────────────────────────────────────┐
│                                                         │  顶部空白
│                                                         │
│                   [Firefox Logo]                        │  Logo（可选，简洁版没有）
│                                                         │
│     ┌───────────────────────────────────────┐         │
│     │  🔍  搜索网络...                      │         │  搜索框
│     └───────────────────────────────────────┘         │
│                                                         │
│  ┌──────┐  ┌──────┐  ┌──────┐  ┌──────┐              │
│  │ 图标  │  │ 图标  │  │ 图标  │  │ 图标  │              │  快捷方式
│  │ 名称  │  │ 名称  │  │ 名称  │  │ 名称  │              │ （6-8个网格）
│  └──────┘  └──────┘  └──────┘  └──────┘              │
│                                                         │
│  ┌───────────────────────────────────────────┐        │
│  │  推荐内容 / Pocket 推荐                    │        │  推荐区（可选）
│  └───────────────────────────────────────────┘        │
│                                                         │
│                                                         │  底部空白
└─────────────────────────────────────────────────────────┘
```

Firefox 主页特点：
- 简洁、干净、留白多
- 搜索框居中，圆角设计
- 快捷方式采用网格布局（每行 6-8 个）
- 深色/浅色主题支持
- 整体色调偏柔和（紫/灰系）

## 改造方案

### 目标布局（Firefox 风格）

```

│  └──────┘  └──────┘  └──────┘  └──────┘  └──────┘   │
│                                                         │
│                                                         │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 番茄钟融入方案

1. **作为快捷方式卡片**：番茄钟是 10 个快捷方式中的一个
   - 图标：Timer
   - 名称：番茄钟
   - 点击时可能显示状态（hover 或常驻小角标）

2. **搜索框内提示**：搜索框 placeholder 可以显示番茄钟状态（可选）

3. **右下角小徽章**：简洁的状态显示（可选）

4. **主要方式**：作为快捷方式卡片，融入网格中，不突兀

### 详细设计

#### 1. 整体风格
- Firefox 风格的深色/浅色主题
- 简洁留白，内容居中
- 紫色系为主色调（Firefox 品牌色）

#### 2. Logo 区域
- "番茄搜索" 文字 + 图标
- 居中显示，简洁风格

#### 3. 搜索框
- 圆角设计（较大圆角）
- 左侧搜索图标
- 居中 placeholder 提示文字
- 聚焦时有边框高亮效果
- 宽度适中（约 600-700px）

#### 4. 快捷方式区域
- 标题："快捷方式"
- 网格布局（每行 5 个，共 2 行 10 个）
- 每个快捷方式：
  - 圆形/圆角方形图标
  - 下方文字名称
  - hover 时有缩放或背景变化
- 快捷方式列表：
  1. 新闻
  2. 天气
  3. 地图
  4. 视频
  5. 购物
  6. 音乐
  7. **番茄钟** ← 番茄钟在这里
  8. 翻译
  9. 邮箱
  10. 设置

#### 5. 番茄钟快捷方式特色
- 图标带有状态指示（小圆点）
  - 绿色：专注中
  - 黄色：休息中
- hover 时显示剩余时间小提示
- 颜色根据状态变化

## 验证步骤

1. 整体布局是否符合 Firefox 新标签页风格
2. 番茄钟是否自然融入快捷方式中
3. 搜索框样式是否简洁美观
4. 快捷方式网格布局是否整齐
5. TypeScript 类型检查: `npx tsc --noEmit`

## 注意事项

1. **风格统一**：严格参照 Firefox 新标签页的简洁风格
2. **番茄钟融入**：作为快捷方式卡片，不再是独立的大卡片
3. **配色**：紫色系为主（Firefox 品牌色）
4. **留白**：要有足够的留白，不要太拥挤
5. **圆角**：搜索框和快捷方式都要有圆角设计
