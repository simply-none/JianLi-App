# 代码编辑模式布局修复计划

## 需求概述

修复代码编辑模式（codeEditorTheme.vue）的布局问题：
1. 左侧目录栏和代码区当前是上下排布，应改为左右排布
2. 需要更严格地参照 VSCode 编辑器的主题设计

## 当前状态分析

### 现有实现（codeEditorTheme.vue）

当前布局结构：
```
┌─────────────────────────────────────┐
│  顶部标签栏 (tab-bar)                │  ← 正确
├─────────────────────────────────────┤
│  侧边栏 (sidebar)                   │  ← 问题：上下排布
├─────────────────────────────────────┤
│  编辑器主区域 (editor-main)          │  ← 问题：应该是右侧
├─────────────────────────────────────┤
│  底部状态栏 (status-bar)             │  ← 正确
└─────────────────────────────────────┘
```

### 问题分析

1. **布局排布错误**：侧边栏和编辑器区域应该是左右排布，不是上下排布
2. **缺少 VSCode 元素**：
   - 缺少活动栏（Activity Bar，左侧图标栏）
   - 底部应该是面板区域（Terminal、Problems 等）+ 状态栏
3. **番茄钟融入**：番茄钟信息应该自然融入当前布局中

## 改造方案

### 目标布局

```
┌────────────────────────────────────────────────────────────┐
│  番茄钟: 专注中 | 剩余: 25:00 | 进度: 65%    [VSCode 标题]│  ← 番茄钟融入顶部
├────┬──────────────────────────────────────────────────────┤
│ 活 │  pomodoro.ts  status.json                    14:30 │  ← 标签栏
│ 动 ├──────┬───────────────────────────────────────────────┤
│ 栏 │      │  ┌──────┐  ┌────────────────────────────┐   │
│    │  📁  │  │行号  │  │  // Pomodoro Timer Status   │   │
│ 📁 │      │  │      │  │  const status = 'work';     │   │
│ 🔍 │  文件 │  │ 1    │  │  const remainingTime = ... │   │
│ 🧩 │      │  │ 2    │  │  const progress = 65;       │   │
│ 🐛 │      │  │ 3    │  │  ...                         │   │
│ ⚙️ │      │  └──────┘  └────────────────────────────┘   │
│    │      ├───────────────────────────────────────────────┤
│    │      │  ┌────────────────────────────────────┐    │
│    │      │  │  TERMINAL                            │    │
│    │      │  │  > npm run build                    │    │
│    │      │  │  > 专注时间剩余 25:00                │    │  ← 番茄钟融入终端
└────┴──────┴──────────────────────────────────────────────┘
│  main  ●Working  Ln 1, Col 1  TypeScript  UTF-8  100%    │  ← 状态栏
└───────────────────────────────────────────────────────────┘
```

### 番茄钟融入方案

1. **顶部标签栏**：番茄钟状态信息作为标签栏右侧的一部分
2. **终端面板**：番茄钟时间作为终端输出的一部分
3. **状态栏**：工作/休息状态作为状态栏左边的指示器
4. 不再是独立的番茄钟小组件，而是融入编辑器生态

### 详细实现

#### 1. 整体布局（Flexbox）
- 容器：水平布局
- 左侧：活动栏（宽度固定 48px）
- 中间主区域：垂直布局
  - 顶部：标签栏
  - 中间：编辑器区域（行号 + 代码内容）
  - 底部：面板（终端）
- 底部：状态栏

#### 2. 活动栏（Activity Bar）
- 宽度：48px
- 图标垂直排列：Explorer、Search、Extensions、Debug、Settings
- 当前激活项有左侧高亮条
- Explorer 激活时显示左侧文件树

#### 3. 文件资源管理器
- 标题：EXPLORER
- 文件夹：src
- 文件列表：pomodoro.ts、status.json
- 当前文件高亮

#### 4. 编辑器区域
- 标签栏：显示打开的文件
- 行号区域
- 代码内容区：带语法高亮的代码
- 代码内容展示番茄钟相关的配置变量

#### 5. 终端面板
- 标题：TERMINAL
- 模拟命令输出
- 番茄钟信息作为命令输出的一部分

#### 6. 状态栏
- Git 分支
- 工作/休息状态指示器
- 光标位置
- 文件类型
- 编码
- 缩进
