# 搜索引擎模式百度风格改造计划

## 需求概述

将当前搜索引擎模式（searchEngine.vue）改为严格模拟百度搜索首页的布局模式，同时让番茄钟自然融入，不再显眼突兀。

## 百度首页布局分析

### 百度首页标准布局

```

│                                                          │
│                                            设置  登录   │  ← 右上角
│                                                          │
│                                                          │
│                                                          │
│                ╔══════════════════╗                     │
│                ║   百度一下，你就知道  ║                     │  ← 大 Logo
│                ╚══════════════════╝                     │
│                                                          │
│         ┌──────────────────────────────┐  ┌──────┐    │
│         │ 🔍 搜索框 (输入内容)         │  │百度一│    │  ← 搜索区
│         │ (搜索建议/历史下拉)          │  │ 下  │    │

└──────────────────────────────────────────────────────────┘
```

### 百度首页特点

1. **顶部导航**：左上角文字链接导航，右上角设置/登录
2. **Logo**：居中大尺寸，百度标志性红蓝色
3. **搜索框**：
   - 圆角/直角（最新版是圆角）
   - 左侧搜索图标
   - 右侧百度一下按钮（蓝色）
   - 下方可出现搜索建议
4. **整体留白**：大量留白，内容居中
5. **配色**：白色背景 + 蓝色主题色 + 红色 Logo
6. **底部**：版权信息和链接，灰色小字

```
└──────────────────────────────────────────────────────────┘
```

### 番茄钟融入方案

1. **搜索框内提示**：
   - 当搜索框为空时，placeholder 显示番茄钟状态
   - 例如："搜索一下  ·  专注中 剩余 25:00"
   - 既展示了状态，又不突兀

2. **顶部导航栏**：
   - 右上角"登录"旁边添加一个番茄钟小图标 + 时间
   - 简洁的状态显示
   - 例如："🍅 25:00  设置  登录"

3. **搜索框下方快捷入口**：
   - 搜索框下方有一行快捷入口
   - 其中一个是"番茄钟"
   - 点击可展开/收起详情

4. **主要方案**：
   - 右上角状态栏：简洁的番茄钟状态（图标 + 时间）
   - 搜索框 placeholder：融合状态提示
   - 不使用独立的大卡片

### 详细设计

#### 1. 顶部导航栏
- **左侧**：新闻、hao123、地图、视频、贴吧、图片、更多 »
- **右侧**：番茄钟状态、设置、登录
  - 番茄钟：图标 + 剩余时间 + 状态颜色
  -  hover 时显示详情

#### 2. Logo 区域
- "番茄搜索" 大字（彩色 Logo）
- 居中显示
- 番茄图标 + 文字
- 字体大小适中

#### 3. 搜索框
- 大尺寸搜索框（约 600-650px 宽）
- 左侧搜索图标
- 输入框（placeholder 显示番茄钟状态）
- 右侧"番茄一下"按钮（蓝色背景，白色文字）
- 圆角设计（百度最新版）

#### 4. 搜索建议（可选）
- 搜索框下方
- 显示几条搜索建议
- 番茄钟相关的建议可以有特殊标记

#### 5. 底部版权
- 关于番茄搜索、用户协议、隐私政策、帮助中心
- 版权信息
- 灰色小字，居中

#### 6. 番茄钟融入细节
- **右上角状态栏**：
  - 🍅 25:00（专注时红色，休息时绿色）
  - 字体小，不显眼
  - hover 时显示完整状态

- **搜索框 placeholder**：
  - 平时："搜索一下你想知道的"
  - 也可以融合状态，可选

## 验证步骤

1. 整体布局是否符合百度搜索首页风格
2. 番茄钟是否自然融入右上角状态栏
3. 搜索框样式是否接近百度
4. 留白和比例是否协调
5. TypeScript 类型检查: `npx tsc --noEmit`

## 注意事项

1. **严格模拟百度**：布局、配色、比例都要贴近百度首页
2. **番茄钟融入**：右上角状态栏 + 搜索框提示，不再用大卡片
3. **配色**：白色背景 + 蓝色按钮 + 红色点缀
4. **留白**：要有足够的留白，内容居中
5. **简洁**：不要添加太多额外元素，保持百度首页的简洁感
